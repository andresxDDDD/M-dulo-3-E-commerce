# M-dulo-3-E-commerce
Desarrollo ejercicio final modulo 3  FUNDAMENTOS DE PROGRAMACIÓN EN PYTHON

Una aplicación de consola desarrollada en Python que simula el flujo básico de una tienda electrónica. Este proyecto fue creado como parte del Módulo 3, aplicando conceptos fundamentales de programación estructurada.

📋 Propósito del Proyecto
Desarrollar un Producto Mínimo Viable (MVP) que permita gestionar un catálogo de productos, realizar búsquedas, administrar un carrito de compras y calcular subtotales y totales de manera dinámica.

🚀 Funcionalidades Principales
Catálogo Dinámico: Visualización de productos con ID único, categoría y precio.

Búsqueda Inteligente: Filtro de productos por nombre o categoría (insensible a mayúsculas).

Gestión de Carrito:

Agregar productos mediante ID con validación de existencia.

Control de cantidades (solo valores positivos).

Vaciado completo del carrito con confirmación de usuario.

Cálculo de Totales: Resumen detallado de la compra incluyendo ID, nombre, cantidad, precio unitario y subtotal por ítem.

🛠️ Tecnologías y Conceptos Aplicados
El desarrollo se limitó estrictamente a las herramientas del Módulo 3 para demostrar dominio de:

Estructuras de Datos: Listas de diccionarios para el catálogo y el carrito.

Lógica de Control: Ciclos while para el menú principal y for para el procesamiento de datos.

Validaciones: Uso de condicionales if/elif/else para entradas de usuario y manejo de errores.

Modularización: Funciones con paso de parámetros y retorno de valores para un código limpio y reutilizable.
